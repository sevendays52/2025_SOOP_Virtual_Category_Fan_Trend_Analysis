# 2025년 SOOP 버튜버 팬 추이 분석
# 2025 SOOP Virtual Category Fan Trend Analysis

## 📊 프로젝트 개요

본 분석은 **단순한 순위 매기기가 아닌, 실제 후원하는 PU(Paying User) 및 큰손(Whale) 시청자의 페르소나를 메트릭스 구조로 분석**하는 것을 목표로 합니다. 스트리머 순위보다는 팬들의 행동 패턴, 충성도, 소비 특성을 파악하여 SOOP 버추얼 스트리밍 생태계를 이해합니다.

## 🔍 핵심 발견

### 페르소나 분류 (총 418명 고유 PU)

| 페르소나 | 인원 | 비율 | 주요 특징 |
|---------|------|------|----------|
| **Whale (20K+)** | 34명 | 8.1% | 고액 후원자, **88.2%가 1회성** |
| **Dolphin (5-20K)** | 105명 | 25.1% | 중간 소비층 |
| **Minnow (<5K)** | 279명 | 66.7% | 소액 후원자 |
| **Loyal (3개월+)** | 1명 | 0.24% | **단 1명뿐!** |
| **멀티오시 (2+)** | 873명 | 19.9% | IP 전환 핵심 타겟 |
| **1회성** | 387명 | 92.6% | 이벤트 집중 후원 |

### 핵심 인사이트

1. **Whale의 88.2%가 1회만 후원** → 생일, 기념일 등 이벤트 집중
2. **Loyal 팬은 418명 중 단 1명** (0.24%) → 장기 충성도 구축 실패
3. **19.9%가 여러 스트리머 지원** → IP 중심 전략 기회
4. **시장 58.2% 하락** (3월→12월) → 이벤트 의존성 증명
5. **PU-RR 0.8%** → 월간 재방문율 극히 낮음

## 💡 전략적 제안

### 1. Whale → Loyal 전환
- **현재:** 88.2%의 1회성 Whale
- **목표:** 이벤트 중심 Whale을 정기 후원자로 전환
- **방법:** 이벤트 후 팔로우업, 멤버십 혜택 제공

### 2. 멀티오시 활용
- **현재:** 19.9% (873명)가 여러 스트리머 지원
- **목표:** IP 기반 크로스 프로모션
- **방법:** 안정적 스트리머 유닛 결성, 그룹 멤버십

### 3. IP 브랜드 구축
- **현재:** 개인 크리에이터 의존
- **목표:** 홀로라이브처럼 그룹 IP
- **방법:** 기업 브랜드 > 개인 인격

### 4. 이벤트 → 구독 모델
- **현재:** 92.6% 1회성 후원
- **목표:** 월간 정기 수익
- **방법:** 구독 티어, 정기 독점 콘텐츠
